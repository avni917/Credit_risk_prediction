# Data

This folder contains the datasets used in the Credit Risk Classification project.

## Files

* **raw_dataset.zip** – Original dataset (compressed).
* **Credit_dataset.csv** – Dataset after preprocessing and feature engineering.

## Data Preprocessing

The cleaned dataset was generated by performing:

* Missing value handling
* Invalid value correction
* Categorical feature encoding
* Multi-label encoding for loan types
* Feature engineering
* Outlier treatment
* Feature scaling (performed after train-test split)

The complete preprocessing workflow is available in the notebook:

`credit risk prediction.ipynb`
